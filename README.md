### О проекте
Этот проект — учебное SPA‑приложение интернет‑магазина на React с каталогом товаров, страницей товара, корзиной и личным кабинетом пользователя. Приложение демонстрирует работу с маршрутизацией, контекстами, локальным хранилищем и адаптивной вёрсткой.

### Основные возможности
* Просмотр каталога товаров по категориям

* Поиск и фильтрация товаров

* Страница детального просмотра товара с галереей изображений

* Добавление товаров в корзину и изменение количества

* Подсчёт итоговой стоимости заказа

* Страница аккаунта с сохранением профиля в localStorage

* Переключение светлой и тёмной темы

* Встроенный онлайн‑чат (Tawk.to)
